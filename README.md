#  Smart Jaggery AI Marketplace

An AI-powered e-commerce platform for jaggery products with Machine Learning capabilities such as demand forecasting, product recommendation, quality prediction, and chatbot integration.

---

## Project Overview

This project is a full-stack **Flask-based e-commerce system** enhanced with **Machine Learning models** to provide intelligent business insights and user experience improvements.

It includes:
- 🛒 E-commerce functionality (cart, orders, wishlist)
- 🤖 AI chatbot for user queries
- 📊 Demand forecasting using ML
- 🌾 Jaggery quality prediction system
- 🎯 Product recommendation engine
- 📈 Admin analytics dashboard

---

## AI/ML Features

### 1️⃣ Product Recommendation System
- Uses similarity-based filtering (Cosine Similarity)
- Suggests similar jaggery products to users

###  Jaggery Quality Prediction
- Model: Random Forest Classifier
- Predicts quality grade (A / B / C)
- Based on: color, moisture, hardness

###  Demand Prediction (Seasonal Forecasting)
- Model: Time-Series Forecasting (Prophet / Regression)
- Predicts future product demand
- Helps in inventory planning

###  AI Chatbot
- NLP-based chatbot using TF-IDF + Naive Bayes
- Answers user queries about products and jaggery
